##使用技术
spring mvc (annotation/ioc),springjdbc + Mysql,Mongodb   

jsp,jstl,el,html,bootstrap,curl,jquery,backbone,underscore   

##摘要    
1.布局采用Bootstrap模版,服务器端采用Spring mvc,数据库采用Mysql和Mongodb。   
2.使用demo的方式记录学习内容。
